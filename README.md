# 💪 FitRanker(競争型ワークアウト記録アプリ)

## 💡 1. プロジェクト概要

FitRanker
は、モチベーション維持を目的としたワークアウト記録アプリケーションです。記録したトレーニング内容に基づき自動で「ポイント」を計算し、それを含めた過去のトレーニング履歴をわかりやすく表示します。
本READMEファイルにAPI側の情報も記載しております。([API側のリポジトリ](https://github.com/Ryosuke-Nanaumi/fitranker-api))

### キーコンセプト

- ポイント制: ワークアウトのレップ数に応じてポイントを獲得。
- 過去の自分のトレーニング履歴を確認可能: 登録したトレーニング情報を後から見直すことができる

## 🚀 4. 開発環境のセットアップ

バックエンドのセットアップは、ルートの makefile に記載された以下のコマンドで行います。
各コマンドの説明は以下です。

| コマンド                | 実行内容                 | 
|---------------------|----------------------|
| `make run`          | 開発用サーバーの起動           |
| `make migrate-up`   | マイグレーション, シードファイルの実行 |
| `make migrate-down` | 一つ前のマイグレーション取消       |

### 4-1. 初期セットアップ (初回のみ)

```
# 1. データベースの作成
PostgreSQL にアプリ用データベースを作成
psql -U postgres
CREATE DATABASE fitranker;
\q

# 3. データベースのセットアップと初期データ投入
# (マイグレーション -> シード実行を連続で行う)
meke migrate-up
```
